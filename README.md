A simple note taking app similar to OneNote.

## Features

* Has multiple different levels of container to manage and organize many notes and collections
* Allows typing and placing uploaded files anywhere
	* file uploads and blocks of text are collapsible to save space
	* allows images, videos, audio, everything
* Allows you to draw to your heart's content
* Allows text styling
